# CartGalaxy : E-Commerce Site

### References 

1. Make a database named "cartgalaxy" in your MySQL workbench.
2. Set enviroment variables such as user_email and passkey.
3. Run http://localhost:8080/index.html on the server to check api's work smoothly.
4. Check http://localhost:8080/swagger-ui/index.html#/ to know about the all api's.
